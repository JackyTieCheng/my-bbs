页面访问
请求路径 more/版块Id-版块帮助部分Id/帮助分类Id-.htm
请求方式 GET
请求参数 参数表 加*号表示参数在URL路径部分
返回值 帮助分页 变量：more_help_page
类型：PageView
分页 PageView
变量 类型 说明
records List<Help> 数据展示列表
pageIndex PageIndex 页码索引
totalpage long 总页数
maxresult long 每页显示记录数
currentpage long 当前页
pagecount long 页码显示总数
onUrl String 上一页URL
nextUrl String 下一页URL
分页索引 PageIndex
变量 类型 说明
startindex long 分页开始索引
endindex long 分页结束索引
帮助 Help
变量 类型 说明
id Long 帮助Id
name String 帮助名称
userName String 用户名称
helpTypeId Long 帮助分类Id
content String 帮助内容
times Date 发表时间
请求参数
参数 类型 是否必填 说明
helpTypeId Long 否 *帮助分类Id
page Integer 否 页码
