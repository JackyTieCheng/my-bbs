页面访问
请求路径 user/control/systemNotifyList.htm
请求方式 GET
请求参数 页码 变量：page
类型：Integer
是否必填：否
返回值 订阅系统通知分页 变量：pageView
类型：PageView
页面访问 - AJAX方式
请求路径 user/control/systemNotifyList.htm
请求方式 GET
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 页码 变量：page
类型：Integer
是否必填：否
请求成功返回 JSON格式值 类型:PageView 值内容说明：订阅系统通知分页
分页 PageView
变量 类型 说明
records List<SubscriptionSystemNotify> 数据展示列表
pageIndex PageIndex 页码索引
totalpage long 总页数
maxresult long 每页显示记录数
currentpage long 当前页
pagecount long 页码显示总数
onUrl String 上一页URL
nextUrl String 下一页URL
分页索引 PageIndex
变量 类型 说明
startindex long 分页开始索引
endindex long 分页结束索引
订阅系统通知 SubscriptionSystemNotify
变量 类型 说明
id String Id
systemNotifyId Long 系统通知Id
userId Long 用户Id
content String 通知内容；本字段在存入数据库时已进行安全过滤，可以加上<#noescape></#noescape>标签关闭转义，以输出html标签
status Integer 消息状态 10:未读 20:已读
sendTime Date 发送时间
readTime Date 阅读时间

示例

提交 删除系统通知 - 默认方式
请求路径 user/control/deleteSystemNotify.htm
请求方式 POST
请求参数 参数表
提交成功返回 默认跳转到jump模板页显示提交成功信息；跳转路径参数jumpUrl有值时按参数值路径跳转
提交失败返回 跳转到message模板页显示提示信息
提交 删除系统通知 - AJAX方式
请求路径 user/control/deleteSystemNotify.htm
请求方式 POST
自定义请求头 协议头：X-Requested-With 值：XMLHttpRequest
请求参数 参数表
提交返回 JSON格式值 类型:Map<String,Object> 键参数：success
值内容说明：成功返回true 失败返回false
值类型：String
键参数：error
值内容说明：错误值集合
值类型：Map<String,String>
删除系统通知 错误值 error
变量 类型 说明
systemNotify String 系统通知错误
token String 令牌错误
提交'删除系统通知'参数
参数 类型 是否必填 说明
subscriptionSystemNotifyId String 是 订阅系统通知Id
token String 是 令牌
jumpUrl String 否 '提交表单 - 默认方式'提交成功后跳转路径,不填写默认跳转到表单提交页；参数必须经过encodeBase64URL编码后才能提交。例${encodeBase64URL('/index')}
